## Creating Demo GIFs

### Requirements

- [asciinema](https://asciinema.org/) for terminal recordings
- [agg](https://github.com/asciinema/agg) for converting to GIF
- [Peek](https://github.com/phw/peek) for screen recordings (Linux)
- [LICEcap](https://www.cockos.com/licecap/) for screen recordings (macOS/Windows)

### Recording Guidelines

1. **Duration**: 6-8 seconds max
2. **Resolution**: 800x600 or 1280x720
3. **Frame Rate**: 10-15 FPS for smaller file size
4. **Content**: Show the key interaction (request → response)

### BankChurn Demo Script

```bash
# Start the service
docker-compose -f docker-compose.demo.yml up -d bankchurn

# Make a prediction request
curl -X POST "http://localhost:8001/predict" \
  -H "Content-Type: application/json" \
  -d '{"CreditScore": 650, "Geography": "France", "Gender": "Female", "Age": 40, "Tenure": 3, "Balance": 60000, "NumOfProducts": 2, "HasCrCard": 1, "IsActiveMember": 1, "EstimatedSalary": 50000}'
```

### CarVision Demo Script

1. Open http://localhost:8501 (Streamlit)
2. Navigate to "Price Predictor" tab
3. Fill in vehicle details
4. Click "Predict Price"
5. Show the result

### TelecomAI Demo Script

```bash
curl -X POST "http://localhost:8003/predict" \
  -H "Content-Type: application/json" \
  -d '{"calls": 50, "minutes": 500, "messages": 100, "mb_used": 20000, "is_ultimate": 0}'
```

## Screenshot Guidelines

- Use consistent browser window size (1280x800)
- Include browser URL bar for context
- Annotate key features with arrows/boxes if helpful
- Save as PNG for quality, compress with `optipng`
