# REST API Reference

All projects expose REST APIs via FastAPI with automatic OpenAPI documentation.

## Common Endpoints

All APIs share these standard endpoints:

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/health` | GET | Health check |
| `/docs` | GET | Swagger UI |
| `/redoc` | GET | ReDoc UI |
| `/openapi.json` | GET | OpenAPI spec |
